## Overview

The Astralform plugin lets you manage AI agent projects directly from Claude Code - create projects, configure LLMs, add MCP servers, and integrate the iOS SDK without leaving your terminal.

## Prerequisites

- [Claude Code CLI](https://claude.ai/code) installed
- Node.js 18+
- Astralform account ([sign up](https://astralform.ai))

## Install the Plugin

```bash
# Clone the plugin
git clone https://github.com/astralform-ai/plugins.git ~/.claude/plugins/astralform

# Build the MCP server
cd ~/.claude/plugins/astralform/mcp-server
npm install
npm run build
```

## Enable the Plugin

Add to your Claude Code settings (`~/.claude/settings.json`):

```json
{
  "plugins": [
    "~/.claude/plugins/astralform"
  ]
}
```

Or enable via Claude Code:

```
/plugins enable ~/.claude/plugins/astralform
```

## Verify Installation

```
/astralform-login
```

You should see a device code and URL to authenticate.
