## Overview

The Astralform plugin lets you manage AI agent projects directly from Claude Code - create projects, configure LLMs, add MCP servers, and integrate the iOS SDK without leaving your terminal.

## Prerequisites

- [Claude Code CLI](https://claude.ai/code) installed
- Node.js 18+
- Astralform account ([sign up](https://astralform.ai))

## Install the Plugin

<Tabs>
  <Tab title="Standard Install (Recommended)">
    First, add the Astralform marketplace to Claude Code:
    ```
    /plugin marketplace add https://github.com/astralform-ai/astralform-plugins
    ```

    Then install the plugin:
    ```
    /plugin install astralform
    ```
  </Tab>
  <Tab title="Direct Install">
    If the marketplace is already added, install directly:
    ```
    /plugin install astralform@astralform-plugins
    ```
  </Tab>
</Tabs>

## Verify Installation

```
/astralform-login
```

You should see a device code and URL to authenticate.
