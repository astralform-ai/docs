## Overview

Astralform is available as a **custom connector** for Claude. Add it once in Claude for web or desktop and Claude can manage your Astralform account directly — teams, projects, API keys, LLM config, connectors, skills, and analytics — through [58 MCP tools](/plugin/mcp-tools).

There is nothing to install and no API key to paste: the connector authenticates with your Astralform account over OAuth, handled by Claude's connector UI.

<Note>
  The connector is the recommended way to use Astralform from Claude. It replaces the former Claude Code plugin and works across every Claude surface, including Claude Code.
</Note>

## Add the connector

<Steps>
  <Step title="Open Connectors">
    In Claude for web or desktop, open **Customize → Connectors**.
  </Step>
  <Step title="Add a custom connector">
    Click the **+** next to Connectors, then **Add custom connector**. Give it a name (for example, `Astralform`) and enter the server URL:

    ```
    https://mcp.astralform.ai/mcp
    ```

    Leave the advanced **OAuth Client ID / secret** fields empty — Astralform registers your client automatically. Click **Add**.
  </Step>
  <Step title="Authorize">
    Claude opens Astralform's sign-in to authorize access. Log in to your Astralform account (or [create one](https://astralform.ai)) and approve. Claude stores the connection — there is no token handling on your side.
  </Step>
  <Step title="Confirm">
    Start a chat and ask:

    ```
    Who am I on Astralform?
    ```

    Claude calls the `astralform_whoami` tool and returns your account identity. You're connected.
  </Step>
</Steps>

## What you can do

Once connected, ask Claude in plain language — it picks the right tool:

- "Create a new Astralform project under my team and set the model to GPT-5.4."
- "List my projects and show the API keys for the production one."
- "Enable the Slack connector on my support agent."
- "What's my token usage this month?"

See the [MCP Tools reference](/plugin/mcp-tools) for the full catalog.

## Authentication & security

- **OAuth, self-serve.** Authorization runs through Astralform's OAuth server; Claude never sees your password and stores only the granted token.
- **Revoke anytime.** Remove the connector from **Customize → Connectors** to revoke access.
- **Scoped to you.** The connector acts as your authenticated Astralform developer account.
