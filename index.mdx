## What is Astralform?

Astralform is an AI agent platform for mobile apps. It provides:

- **iOS SDK** - Native Swift SDK for seamless integration
- **Multi-provider LLM support** - OpenAI, Anthropic, Groq, Ollama
- **Tool calling** - Server-side and client-side MCP tools
- **Streaming responses** - Real-time SSE-based chat streaming
- **Conversation persistence** - Automatic message history management

## Architecture

```
iOS App                      Astralform Backend              LLM Provider
   │                                │                              │
   │──── SSE /v1/chat/stream ──────►│                              │
   │                                │───── LangGraph agent ───────►│
   │                                │                              │
   │◄─── streaming response ────────│◄──── AI response ───────────│
   │                                │                              │
   │◄─── tool_use event ────────────│  (client tool requested)     │
   │                                │                              │
   │──── POST /v1/tool-result ─────►│                              │
   │                                │───── continue agent ────────►│
```

## Tool Execution Layers

Astralform supports three layers of tool execution:

| Layer | Location | Who Configures | Example Tools |
|-------|----------|----------------|---------------|
| **Platform Tools** | Backend | App developer | Tavily search |
| **Server MCP** | Backend | App developer | Database, Slack |
| **Client MCP** | iOS Device | End user | Calendar, Contacts |
